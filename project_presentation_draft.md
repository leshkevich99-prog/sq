# Squadra CRM: Технический драфт для презентации

### 📍 01. Платформа и UX-стратегия
- **База**: Telegram Mini App (PWA внутри экосистемы TG).
- **Вход**: Беспарольный (Telegram Init Data). Один клик — полная авторизация.
- **Инсталляция**: Отсутствует. Работает из любого чата или через бота.
- **Уведомления**: Гибридный Push (In-app + Telegram Bot API). Кнопки прямого перехода (Deep Linking).

### 👥 02. Операционная модель (Роли)
- **Клиент**: Заказ (Pitstop/Телеметрия/Семья), гараж (учет авто), баланс квот, история чеков.
- **Пилот**: Рабочая область, чек-листы задач, учет ГСМ/СТО, фотоотчеты.
- **Админ**: Центр управления (Kanban), назначение исполнителей, финансовый мониторинг сотрудников и клиентов.

### 🗺 03. Модуль «Пилот»: Навигация и Отчетность
- **Smart-навигация**: Автоматический разбор координат из текста. Прямой форвардинг в Яндекс/Google Maps (режим «Маршрут»).
- **Медиа-доказательства**: Параллельная загрузка фото «До/После». Сжатие на лету (экономия трафика, скорость на слабом сигнале).
- **Финансовое поле**: Ввод расходов (топливо, сервис) с прикреплением фото-чека. Мгновенная синхронизация с облаком.

### 📊 04. Модуль «Админ»: Управление и Биллинг
- **Kanban Flow**: 4 этапа контроля (Новые -> В работе -> Проверка -> Готово).
- **Smart Assignment**: Выбор пилота с внутренним комментарием (авто-часа в задачу). Пошаговое подтверждение.
- **Billing**: Формирование счетов, загрузка квитанций, управление квотами клиентов в реальном времени.

### 🧱 05. Технологический стек (Architecture)
- **Database**: Firebase (Firestore для данных + Realtime SDK для чатов и статусов).
- **Frontend**: React + Vite + Tailwind (высокая частота кадров, плавные анимации).
- **Backend**: Node.js (Express) + Telegram Bot API v2.0.
- **Stability**: Логика восстановления сессии при сворачивании (Visibility Change Listener).

### 💎 06. Ключевые преимущества (USP)
- **Zero Friction**: Нет регистрации — нет потерь лидов на входе.
- **Real-time Sync**: Изменение статуса у пилота отображается у клиента и админа за <0.5 сек.
- **Resilience**: Система работает при нестабильном интернете и в фоновом режиме на iPhone/Android.
